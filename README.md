## Getting Started
### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/aleiva17/checker-ai-app.git
   ```
2. Install NodeJS packages in the cloned file
   ```sh
   npm install
   ```
3. Run the following command to run the webpage
   ```sh
   npm run dev
   ```

### Built With
- React JS
- TypeScript
- React Router
- Tailwind CSS (with Dark and Light theme support)
- Vite JS
